# OpenCV practical projects

Collected OpenCV practical projects with c++ and python

## Dependency

- cmake 3.16+

- python3.8+

- opencv 4.5.3/4.5.5

- matplotlib-cpp

- jsoncpp

- png++

- ubuntu 21.10

- gcc-11/g++-11 for C++ 14 and 17
