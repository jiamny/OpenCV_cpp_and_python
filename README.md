# OpenCV practical projects

Collected OpenCV practical projects with c++ and python

## Dependency

- cmake 3.16+

- python3.8

- opencv 4.5.3

- matplotlib-cpp

- jsoncpp

- png++

- ubuntu 20.04

- gcc-9/g++-9 for C++ 14 and 17
